## How to update npm to latest version?

1. Go to the folder with installed nodejs

```bash
cd "C:\Program Files\nodejs"
# or
cd "C:\Program Files (x86)\nodejs"
```

2. Install lattest version of npm

```bash
npm install npm@latest
```
